Мануал по работе с базой Python

Введение в Python

Python — это высокоуровневый язык программирования, который легко учить и читать. Он используется для разработки веб-приложений, анализа данных, машинного обучения, автоматизации процессов и многих других задач.

1. Основы синтаксиса

1.1 Переменные

age = 25               # Целое число
name = "Alice"        # Строка
is_student = True     # Логическое значение

print(age, name, is_student)

2. Типы данных

2.1 Списки, кортежи и словари

Списки, кортежи и словари — это структуры данных, которые позволяют хранить коллекции значений.

# Списки
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")  # Добавляем элемент в список
print(fruits)

# Кортежи
colors = ("red", "green", "blue")
print(colors[1])  # Доступ к элементу по индексу

# Словари
person = {
    "name": "Bob",
    "age": 30,
    "city": "New York"
}
print(person["name"])  # Доступ к значению по ключу

3. Операции с переменными и встроенные функции

# Примеры использования встроенных функций
print(len(fruits))         # Количество фруктов в списке
print(type(name))          # Тип переменной name
print(str(age))            # Преобразуем число в строку

4. Условия и циклы

4.1 Условные операторы

if age < 18:
    print("You are a minor.")
else:
    print("You are an adult.")

4.2 Циклы

# Цикл for
for fruit in fruits:
    print(fruit)

# Цикл while
count = 0
while count < 3:
    print("This is iteration", count)
    count += 1

5. Основные рекомендации по работе с окружением

Python можно установить на компьютере с помощью дистрибутивов, таких как Anaconda или с помощью установки с сайта Python.
